À propos

Cette application Angular affiche un tableau des médailles olympiques par pays. Elle utilise un fichier medals.json côté client et des composants standalone.

  • • Angular Standalone Components
  • • Bootstrap 5 (CSS)
  • • HttpClient pour charger les données JSON
  • • Routage (Home / À propos)
